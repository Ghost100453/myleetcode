# 56. 合并区间

> ghost 2020年7月12日
>
> 难度：==中等==

## 题目描述

给出一个区间的集合，请合并所有重叠的区间。

示例 1:

```
输入: [[1,3],[2,6],[8,10],[15,18]]
输出: [[1,6],[8,10],[15,18]]
解释: 区间 [1,3] 和 [2,6] 重叠, 将它们合并为 [1,6].
```


示例 2:

```
输入: [[1,4],[4,5]]
输出: [[1,5]]
解释: 区间 [1,4] 和 [4,5] 可被视为重叠区间。
```

## 解题思路

将所有的区间按照左区间进行排序（升序），将第一个区间作为基准，如果之后的区间，其左区间的值小于第一个区间的右区间，则进行合并，否则，将其作为第二个区间加入，依次处理即可。
